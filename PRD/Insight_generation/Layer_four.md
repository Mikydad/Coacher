🚀 LAYER 4 — DELIVERY ENGINE (VERY IMPORTANT)
🎯 Purpose

Decide:

Which insight to show, when, and how

Because even perfect insights are useless if:

shown at the wrong time
shown too often
shown in the wrong place
🧠 Responsibilities
1. Insight Selection

Pick:

1 primary insight
optional 1 secondary
2. Timing Rules

Examples:

Morning → show “do now”
Evening → show “reflection”
After completion → show reinforcement

And the AI might report to the user right before sleep to help the user have a positive self love of confidence.
3. Suppression Rules
don’t repeat same insight within 24h
don’t show low-confidence insights
don’t interrupt active task flow
4. Surface Routing

Where it goes:

home card (primary)
notification (only high urgency)
progress screen 
🧠 Output
{
  "selectedInsight": "...",
  "surface": "home",
  "priority": "high",
  "shouldNotify": true
}
🔥 Why Layer 4 matters

This is what makes your app feel like:

“It knows when to speak and when to stay quiet”
